Full Stack Projects

Next Round's On Me

Social network where users can follow friends, check out local bars and restaurants, send friends 'rounds', and redeem 'rounds' sent to them. Architecture uses React / Redux, Express, Node.js, and PostgreSQL. Includes AWS for image hosting. All styling uses raw CSS3 to imitate Facebook styling (circa late 2020).

Codevous

Meetup clone focused on programmers. Architecture uses React / Redux, Flask, SQLAlchemy, and PostgreSQL. All styling uses raw CSS3 to give a code editor / terminal feel.

Bike Around

AirBnB clone focused on renting bicycles. Architecture uses React / Redux, Express, Node.js, and PostgreSQL.

What to Watch

Goodreads clone for movies. Architecture uses Pug, Express, Node.js, and PostgreSQL.
